## 🚀 Features

- 🎯 Difficulty Levels (Easy / Medium / Hard)  
- ⏱ Countdown Timer System  
- 🧠 Smart Hints (Close / Far / Very Close)  
- 📊 Guess History Tracking  
- 🏆 Score & High Score System  
- 💾 Data Persistence (JSON Save/Load)  
- 🌙 Dark / Light Mode Toggle  
- 🎮 Start Menu Interface  
- ⌨️ Keyboard Support (Enter Key)  

---

## 🛠️ Tech Stack

- **Language:** Python 🐍  
- **GUI Library:** Tkinter  
- **Data Storage:** JSON

## 📌 Project Details

<details>
<summary>🎮 How to Play</summary>

- Select difficulty level  
- Start the game from menu  
- Enter your guess number  
- Receive smart hints after each attempt  
- Try to guess the correct number before timer ends  
- Final score is based on attempts and time  

</details>

<details>
<summary>📈 What I Learned</summary>

- Building GUI applications using Tkinter  
- Object-Oriented Programming (OOP) design  
- Managing game state (timer, score, difficulty)  
- Working with JSON for saving/loading data  
- Improving UI/UX for better user experience  

</details>

<details>
<summary>⚠️ Challenges Faced</summary>

- Managing multiple timers at once  
- Handling global variables causing messy code  
- Keeping UI responsive during gameplay  
- Structuring code for future scalability  

</details>

<details>
<summary>💡 Solutions</summary>

- Converted project into class-based architecture  
- Used `after()` method properly for timer control  
- Separated UI and logic into different sections  
- Improved code readability and modular structure  

</details>

<summary>🔮 Future Improvements</summary>
